# RailPway_Technical

## Basic Technical Reference for Railway Permanent Way Maintenance

This repository contains the source code, templates, engineering diagrams,
illustrations and publishing framework used to generate the

**Basic Technical Reference for Railway Permanent Way Maintenance**

The handbook is intended for:

- Railway Engineers
- Permanent Way Inspectors
- Industrial Railway Sidings
- IOCL Maintenance Engineers
- Contractors
- Training Institutes
